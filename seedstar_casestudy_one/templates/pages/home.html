Welcome to the Email Storage Application

ASSUMPTIONS

Below is a list of the assumptions made on this project

  • A user of the application should only have one email address
  • Admin should be able to access the data through the default django admin dashboard
  • Email addresses and their corresponding name will not be associated with any particular user.
  • The application will not handle any media file in production.
  • Email addresss should be unique throughout the system
  • The database vendor will be used in both developemnt and production
  • Scaling up the application should be easy
  • The development environment should not be dependent on the developer's system
  • I assumed that the code must decoupled enough to enable unit testing
